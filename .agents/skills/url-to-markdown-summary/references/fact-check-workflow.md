# Fact Check Workflow

## 目的

既存の `official` / `curation` 記事を原典URLと照合し、本文と管理情報を更新する。

## 確認手順

1. 記事先頭の `原典` と `最終ファクトチェック` を確認する
2. 原典URLの内容と記事本文を見比べる
3. 論点、注意点、具体例が古くなっていないか確認する
4. 差分があれば本文を直す
5. `最終ファクトチェック` を更新する
6. `確認メモ` を短く更新する
7. `docs/operations/article-registry.md` の同じ記事行も更新する

## official の判断基準

- 原典の内容と本文が一致しているか
- 公式にない断定が紛れていないか
- 表現変更や仕様変更が本文に反映されているか

## curation の判断基準

- 原典記事の主張を正しく要約しているか
- 前提条件が変わっていないか
- 必要なら公式情報と矛盾していないか

## メモの例

- 原典更新なし
- 見出し構成のみ調整
- 公式側の表現変更を反映
- 外部記事の前提が古いため注意書きを追加
