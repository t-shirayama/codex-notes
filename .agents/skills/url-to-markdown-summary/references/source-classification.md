# Source Classification

## Official

- 公式ドキュメント
- 公式ブログ
- 公式ヘルプセンター
- 公式GitHubや一次配布資料

扱い方:

- 事実ベースで要約する
- 原典URLを明記する
- 憶測を混ぜない

## Curation

- Qiita
- Zenn
- 個人ブログ
- 登壇レポートや紹介記事

扱い方:

- 実践知見として要点を再構成する
- 公式情報と矛盾する場合は、公式を優先する
- 個人の意見や前提条件をそのまま一般化しない

## Mixed Sources

- 公式と外部記事が混在する場合は、先に公式の事実を整理する
- 外部記事は「実務での補助線」「現場での解釈」として位置づける
- 本文中で、事実と補足が読み分けられるようにする
