# Official Output Format

公式URLを要約するときは、原則として次の構成で書く。

```md
# タイトル

> 出典: [ページ名](URL)

## 概要

このページで何が分かるかを2〜4文で書く。

## 何が分かるか

- 要点1
- 要点2
- 要点3

## 要点

### 1. 主要な考え方

事実ベースで整理する。

### 2. 実務での意味

開発や運用でどう効くかを書く。

### 3. 具体例

短い例を入れる。

## 原典URL

- URL
```

## Notes

- 逐語訳にしない
- 見出し名は内容に合わせて調整してよい
- 公式に書かれていないことは断定しない
