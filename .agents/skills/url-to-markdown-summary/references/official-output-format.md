# Official Output Format

公式URLを要約するときは、原則として次の構成で書く。

```md
# タイトル

> 種別: official
> 参考元: [ページ名](URL)
> 最終ファクトチェック: YYYY-MM-DD
> 確認メモ: 参考元更新なし

## 概要

このページで何が分かるかを2〜4文で書く。

## 何が分かるか

- 要点1
- 要点2
- 要点3

## 要点

### 1. 主要な考え方

事実ベースで整理する。

### 2. 実務での意味

開発や運用でどう効くかを書く。

### 3. 具体例

短い例を入れる。

## 参考元

このページは以下の参考元に基づいて整理しています。未確認の推測は含めていません。

- URL
```

## Notes

- 逐語訳にしない
- 見出し名は内容に合わせて調整してよい
- 公式に書かれていないことは断定しない
