# 環境変数と秘密情報

## 概要

Webアプリで使う環境変数と秘密情報の扱いを決める。

## ルール

- 実トークン、実パスワード、秘密鍵をコミットしない
- 共有するのは `.env.example` の変数名と用途だけにする
- ローカル開発用の値は本番と分ける
- 環境変数を追加したら、`.env.example` とこのファイルを同時に更新する

## 変数例

| 変数名 | 用途 | 必須 | メモ |
| --- | --- | --- | --- |
| `APP_BASE_URL` | アプリのベースURL | yes | ローカルでは `http://localhost:3000` |
| `DATABASE_URL` | DB接続先 | yes | サンプル値のみ |
| `AUTH_SECRET` | 認証用secret | yes | 実値は書かない |
| `EMAIL_PROVIDER_API_KEY` | メール送信 | no | 使う場合だけ設定 |
